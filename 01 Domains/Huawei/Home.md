# Huawei Knowledge Base

## Objectif

Construire une base de connaissance Huawei en français, orientée certification, pratique terrain et production de contenu TianSemi.

Le domaine Huawei TSOS est organisé autour de trois parcours :

1. [[01 Datacom/Home|Datacom]]
2. [[02 Security/Home|Security]]
3. [[03 WLAN/Home|WLAN]]

Cloud n'est pas conservé dans ce domaine Huawei. Les labs ne sont pas un parcours séparé : ils doivent vivre dans chaque certification et chaque module.
