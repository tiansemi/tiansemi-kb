# TianSemi Operating System

Bienvenue dans le **TianSemi Operating System (TSOS)**.

Ce coffre Obsidian est le cerveau de TianSemi. Il centralise :

- la connaissance technique ;
- les laboratoires ;
- les références ;
- les templates ;
- les prompts ;
- les assets ;
- le moteur de publication ;
- la préparation Huawei ICT Competition ;
- la production de contenu pour le site et les réseaux sociaux.

Site cible : http://tiansemi.github.io/
